# Module 3 Assignment

JavaScript Fundamentals Practice

## Topics
- Variables
- Data Types
- Operators
- Conditionals
- Loops

## Files

| File | Task | Description |
|------|------|-------------|
| `student.js` | Task 1 | Store and display student information using variables |
| `datatype.js` | Task 2 | Declare 5 different data types and check with `typeof` |
| `calculator.js` | Task 3 | Perform arithmetic operations on two numbers |
| `result.js` | Task 4 | Determine grade using `if...else if...else` conditions |
| `for-loop.js` | Task 5 | Print numbers 1 to 10 using a `for` loop |
| `while-loop.js` | Task 6 | Print numbers 10 to 1 using a `while` loop |

## How to Run

Make sure [Node.js](https://nodejs.org/) is installed, then run any file with:

```bash
node student.js
node datatype.js
node calculator.js
node result.js
node for-loop.js
node while-loop.js
```
